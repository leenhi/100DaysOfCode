Random Color Generator

Click the "Generate Color" button below to get a background color.

Code Challenge Info:

I created a div with an id="colorBox". This colorBox will generate a random background color everytime the "Generate Color" button is clicked.

- addEventListener() was used to call the function.

- There are 0 - 255 hexidecimal colors for RGB(Red, Green, Blue). I created 3 variables from x, y, z. Each of them gets a random number from Math.round( Math.random() * 255 ).

- Whenever the "Generate Color" button is clicked, the "colorBox" background is equal to 'rgb(' + x + ',' + y + ',' + z + ')';
