Code Challenge Info:

This features a search that "filters" all the names inside the list(li) and displays the name that matched. The names that don't match will be hidden.

- The filter() method creates an array filled with all array elements that pass a test (provided as a function). - w3Schools.com

- The toUpperCase() method converts all the alphabetic characters in a string to uppercase. Both the search "input values" and list item "names" were converted to uppercase so that the "strings" have a higher chance of findind a match.

- The querySelectoryAll() can select anything based on its class and element name.
